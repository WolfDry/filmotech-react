# FilmoTech

Progressive web app qui liste des films et leurs disponibilité dans les cinémas proche

# Fonctionnalités 

- Push notif lors de l'ajout d'un cinéma
- Localisation des cinémas autour de toi
- Synopsis des films en text to speech

# Obligation

- Offline first
- Responsive
